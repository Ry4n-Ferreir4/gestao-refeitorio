## Diagrama de Interação

```mermaid
sequenceDiagram
    participant Usuario as Administrador/Gestor
    participant Sistema as Sistema
    participant Banco as Banco de Dados

    Usuario->>Sistema: +selecionarCriarCardapio()
    activate Sistema
    Sistema->>Usuario: +exibirFormularioCardapio()
    Usuario->>Sistema: +preencherFormulario(nome: string, dataInicio: Date, dataFim: Date, alimentos: List<Refeicao>)
    Sistema->>Sistema: +validarDados(nome: string, dataInicio: Date, dataFim: Date, alimentos: List<Refeicao>)
    alt Dados Válidos
        Sistema->>Banco: +salvarCardapio(nome: string, dataInicio: Date, dataFim: Date, alimentos: List<Refeicao>)
        activate Banco
        Banco-->>Sistema: +confirmacaoSalvar()
        deactivate Banco
        Sistema-->>Usuario: +mensagemSucesso()
    else Dados Inválidos
        Sistema-->>Usuario: +mensagemErroDados()
    end
    alt Erro ao Salvar
        Banco-->>Sistema: +erroSalvar()
        Sistema-->>Usuario: +mensagemErroSalvar()
    end
    deactivate Sistema
